# API testing with [Bruno](https://www.usebruno.com/)

You must edit the `./environments/local-env.yml` file first to have appropriate user and keycloak connection. 

To access any secure endpoint make sure the environment variable is exist via `./auth/get-token.yml`. The API call will provide necessary post script and test.

For more detail about syntax checkout [opencollection docs](https://www.opencollection.com/).
